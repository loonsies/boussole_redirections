# Redirections for boussole

## This repository simply aims to provide custom maps for zones not covered by the variety of DATs mods available online.

## How to install

- Simply copy the content of settings.lua for a given set of maps inside your configuration file inside `<Ashita Folder>/config/addons/boussole/<YourCharacter_XXXXXX>`

- Then copy the content of custom_maps inside `<Ashita Folder>/config/addons/boussole/custom_maps`

- Reload boussole
